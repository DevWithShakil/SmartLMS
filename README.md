# SmartLMS - Laravel Live Test Project

SmartLMS is a mini-project developed to demonstrate core Laravel functionalities, including Query Builder, Eloquent ORM, and REST API integration with Axios.

## 🚀 Features

This project covers three main tasks as per the live test requirements:

### Task 1: Laravel Query Builder & Aggregates

- Fetching all students from the database.
- Filtering students based on marks (marks > 50).
- Calculating the average marks using the `avg()` aggregate function.

### Task 2: Laravel Eloquent ORM

- Performing CRUD operations using the Product model.
- Filtering products based on stock availability.
- Inserting new records and updating existing data (increasing product price).

### Task 3: Laravel REST API + AJAX (Axios)

- Creating a custom GET API endpoint.
- Returning data in a structured JSON format.
- Consuming the API from the frontend (Blade) using **Axios**.

## 🛠️ Technology Stack

- **Framework:** Laravel 11
- **Database:** MySQL
- **Frontend:** Blade Templating Engine
- **HTTP Client:** Axios (AJAX)
